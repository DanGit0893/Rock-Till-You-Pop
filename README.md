## Project Description
> A short description of your game.
My app is called Rockem Sockem Robots. It's a game app for players who want to play Rockem Sockem Robots online. Two players can join an online game and play a round of RSR, with game results displayed at then end of a round -- whichever player won. Users will have the option to play again.

## User Stories

### MVP Goals

- As a player I want my game to recognize and display when one players key count has reached the preset win condition i.e(70 presses)so that I know who won.
- As a player I would like to have both challengers play without issue
- As a player I would like to have one button assigned to me and one assigned to a challenger 
- As a player I would like to be able to visually see who is winning or losing by way of the character growing
- As a player I would like to be able to restart the game after a win or loss


### Stretch Goals

- As a player I would like to be able to play online and not have my challenger sitting next to me
- As a player I would like to be able to play against a computer If I don't have a challenger 
- As a player I would like to see enter a name for each player and have it displayed above their character
- As a player I would like to be able to select an avatar to use as my character
- As a player I would like to know how many keys were hit by each player at the end to know how close the score was
